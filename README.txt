Import wxpython, numpy and OpenCV to execute this program.

set this command on your terminal.
-> wpython color_ga_elite.py

(i)color_ga_elite.py
This python file is main program in this project.
UI and drawing function are include in this file.

(ii)ga.py
GA algorithm is written in this file.
This GA-alogrithm have 2-points crossover and mutation.
Selection method are Roulett and Elite.

(iii)hsv_eval.py
This program evaluate color combination made by color_ga_elite.py.
I use Moon & Spencer's color harmony theory as evaluation value.
